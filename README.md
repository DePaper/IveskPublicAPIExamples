# IveskPublicAPIExamples
Examples of how to use Įvesk.lt PublicAPI

This repository contains examples of how to use various Įvesk.lt Rest API endpoints.

# Requirements
To run these examples, you need:
* Node.js 20.x and up.
* Įvesk.lt API key. You can generate it here: [https://app.ivesk.lt/settings/general](https://app.ivesk.lt/settings/general)
